# System Cache Monitor

A simple background tool to monitor and manage system cache, keeping your Windows environment clean and responsive.

## Features
- Background monitoring using `pythonw`.
- Efficient cache management.
- Lightweight and portable.

## How to Run
1. Ensure you have Python installed.
2. Run `System Cache Monitor.bat` to start the monitor in the background.

## Scripts
- `cache_monitor.py`: The main Python script that handles the monitoring logic.
- `System Cache Monitor.bat`: A launcher script that runs the monitor in the background.
